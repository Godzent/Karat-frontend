# Karat Frontend

React frontend for Karat - Invest your spare change & earn rewards from receipts.

> ⚠️ **MVP NOTICE:** This is a simulated trading environment for testing purposes only. No real money or securities are involved.

## Features

- User authentication (login/signup)
- Receipt upload and tracking
- Portfolio management
- Real-time balance updates
- Mobile-responsive design

## Live API

Connected to: `https://web-production-65513.up.railway.app/api`

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel

1. Push to GitHub
2. Import repo in Vercel
3. Deploy (no config needed!)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide Icons
